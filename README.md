# Streak Tracker

A tiny single-file habit streak tracker. Click the circle next to a habit each day you keep it going, and watch your streak and 30-day heatmap grow.

Built as a first project for learning GitHub Desktop: local commit -> publish -> GitHub Pages.

## Run it
Just open `index.html` in a browser, or visit the live version on GitHub Pages. Data is stored in `localStorage`, so it's private to your browser.
